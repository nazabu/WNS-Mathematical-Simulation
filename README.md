# White-Nose Syndrome (WNS) Mathematical Simulation

### What is this project?
Simulation of bat populations in a single cave affected by White-Nose Syndrome using mathematical models.
This is the code for my research project [name?]. The data, at the moment, is arbitrary, but that doesn't necessarily invalidate the stress analysis.
If you're interested in any part of this project...

### Setup
1. Clone the repository:

`git clone https://github.com/nazabu/WNS-Mathematical-Simulation`

2. Open it with your code editor or run the files through the terminal:

```
cd WNS-Mathematical-Simulation/src
python3 SIRP.py
```

3. Install Poetry (if it's not already installed) and install dependencies:
```
curl -sSL https://install.python-poetry.org | python3 -
poetry install
```
